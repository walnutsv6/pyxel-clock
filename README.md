# pyxel-clock

Pyxelを使用した時計アプリです。  
Pyxelが使用できるハンドヘルド端末での利用を想定しています。

## 基本仕様

- 画面サイズ：144x144
- 現在時刻のアナログ表示およびデジタル表示に対応
- デジタル表示部分を現在時刻／日付切り替え可能
- 現在時刻のデジタル表示は 12 時間表示 / 24 時間表示の切り替えに対応
- タイマー画面でも現在時刻を表示

## 動作環境

- Python 3.10.x以上 (64bit)
- Pyxel 2.9.0

## 操作方法

| ゲームパッド | キーボード | 操作（時計画面） | 操作（タイマー画面） |
|---|---|---|---|
| Aボタン | SPACEキー | 時刻表示 / 日付表示の切り替え | タイマー開始 / 一時停止 |
| Bボタン | Rキー | 12時間表示 / 24時間表示の切り替え | タイマーをリセット |
| ↑ボタン | 上方向キー |  | タイマー時間を1分単位で調整 |
| ↓ボタン | 下方向キー |  | タイマー時間を1分単位で調整 |
| ←ボタン | 左方向キー |  | 時計画面へ戻る |
| →ボタン | 右方向キー | タイマー画面へ移動 |  |
| START＋BACK（SELECT）ボタン | Qキー | アプリケーション終了 | アプリケーション終了 |
