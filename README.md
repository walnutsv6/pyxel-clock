# pyxel-clock

Pyxelを使用した時計アプリです。  
Pyxelが使用できるハンドヘルド端末での利用を想定しています。

## 基本仕様

- 画面サイズ：144x144
- 現在時刻のアナログ表示およびデジタル表示に対応
- デジタル表示部分を現在時刻／日付切り替え可能

## 操作方法

- Aボタン：時刻／日付表示の切り替え
- START＋BACK（SELECT）ボタン：終了
