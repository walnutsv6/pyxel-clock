# Pyxel アナログ時計 仕様書

## 概要

Python の retro ゲームエンジン [Pyxel](https://github.com/kitao/pyxel) を使ったアナログ時計＋タイマーアプリ。

## 起動方法

```bash
pip install pyxel   # 未インストールの場合
python main.py
```

## ファイル構成

```
pyxel_clock/
├── main.py       # エントリポイント。clock.run() を呼び出す
├── clock.py      # App クラス定義、pyxel の初期化・ループ管理
├── timer.py      # タイマー画面の描画
├── face.py       # 文字盤描画（ベゼル、目盛り）
├── hands.py      # 針描画（時針・分針・秒針）
├── utils.py      # 座標変換ヘルパー
├── clock.pyxres  # Pyxel リソースファイル（スプライト・ミュージック）
└── assets/
    └── x8y12pxTheStrongGamer.ttf  # デジタル表示用フォント
```

## 技術仕様

| 項目             | 値                        |
|------------------|---------------------------|
| 画面サイズ       | 144×144 px                |
| 表示スケール     | 4 (→ 576×576 ウィンドウ)  |
| 中心座標         | cx=72, cy=64              |
| 時計半径         | 58 px                     |
| FPS              | 10                        |
| Python バージョン | 3.x                       |
| Pyxel バージョン | 2.x (display_scale 対応)  |

## 画面構成

アプリは **時計画面** と **タイマー画面** の 2 画面で構成される。`→` キーまたはゲームパッドの D-Pad 右でタイマー画面へ、`←` または D-Pad 左で時計画面に戻る。

---

## 時計画面

### 表示内容

- アナログ時計（文字盤＋時針・分針・秒針）
- 画面下部にデジタル表示（時刻または日付）
- タイマーがアクティブな場合、右上にタイマー状態を表示

### キー操作

| キー / ボタン           | 動作                             |
|-------------------------|----------------------------------|
| Q / START+BACK          | アプリ終了                       |
| SPACE / A ボタン        | デジタル表示を時刻 ↔ 日付 切替   |
| R / B ボタン            | 時刻表示を 12h ↔ 24h 切替        |
| → / D-Pad 右            | タイマー画面へ移動               |

### デジタル表示

- **時刻モード**: `HH:MM:SS`（24h）または `AM/PM HH:MM:SS`（12h）
- **日付モード**: `YYYY/MM/DD Ddd`（曜日略称付き）
- フォント: `x8y12pxTheStrongGamer.ttf`（サイズ 12）、文字幅 8px で中央揃え

### 時計画面でのタイマー状態表示

タイマーが動作中・一時停止中・完了のいずれかである場合（`remaining < set_seconds` も含む）、右上にタイマーアイコンと残り時間を表示する。

| 状態             | アイコン表示                         | 文字色       |
|------------------|--------------------------------------|--------------|
| 動作中           | 点滅（0.5秒間隔 = 5フレーム）        | 白 (7)       |
| 一時停止中       | 常時表示                             | グレー (5)   |
| 完了             | 点滅                                 | 赤点滅 (8/0) |
| 残り30秒以下     | （上記に準ずる）                     | 黄色 (10)    |

- アイコン: `pyxel.blt(108, 2, 0, 32, 0, 8, 8, 0)`（スプライトシート座標 (32,0)、8×8px）
- 残り時間テキスト: `MM:SS` 形式、座標 (120, 4)

## タイマー画面

### 表示内容

- 時計と同じアナログ盤面（文字盤＋分針・秒針）
  - **時針は非表示**
  - 12時位置内側に "TIMER" ラベル（グレー）
- 盤面内の中段（y=84）に現在時刻（12h/24h に従う）をグレー表示
- 画面下部に残り時間をデジタル表示（`MM:SS`、フォント付き）
- 右上にタイマーアイコン（常時表示）

### アナログ針の動作

タイマーモードでは `draw_hands(CX, CY, 0, minutes, seconds, show_hour=False)` を呼び出す。
- 分針: 残り時間の「分」
- 秒針: 残り時間の「秒」

### デジタル残り時間の色

| 状態                   | 文字色              |
|------------------------|---------------------|
| 完了（アラーム中）     | 赤点滅 (8/0)        |
| 残り 30 秒以下         | 黄色 (10)           |
| 動作中                 | 白 (7)              |
| 一時停止中             | ピーチ (13)         |
| 開始前（未スタート）   | 薄グレー (6)        |

### キー操作

| キー / ボタン           | 動作                                        |
|-------------------------|---------------------------------------------|
| Q / START+BACK          | アプリ終了                                  |
| SPACE / A ボタン        | スタート / 一時停止（完了後は無効）         |
| R / B ボタン            | リセット（残り時間を設定秒数に戻す）        |
| ↑ / D-Pad 上            | 設定時間 +1分（上限 60分、未実行時のみ）    |
| ↓ / D-Pad 下            | 設定時間 -1分（下限 1分、未実行時のみ）     |
| ← / D-Pad 左            | 時計画面へ戻る（完了中はアラーム停止も行う）|

- 上下キーは長押し対応（初回 10フレーム待ち、以降 3フレームごとに連続入力）
- 設定時間の上限: 60分（3600秒）、下限: 1分（60秒）

### タイマーの動作仕様

- FPS=10 のため、毎フレーム `timer_remaining -= 0.1` でカウントダウン
- 表示上の残り時間は `math.ceil(remaining)` で切り上げ（0になった瞬間にアラームが鳴るように +1秒ズレを解消）
- **時計画面に戻ってもタイマーはバックグラウンドで動作し続ける**（`update()` はページに依存しないカウントダウン処理を含む）
- 残り時間が 0 になると: `timer_running=False`、`timer_done=True`、`pyxel.playm(0, loop=True)` でアラーム再生

## モジュール詳細

### `main.py`

エントリポイント。`clock.run()` を呼び出すだけ。

### `clock.py`

- `App` クラス: `pyxel.init` → `pyxel.load("clock.pyxres")` → `pyxel.run` でループ開始
- `update()`: キー入力処理（終了・画面切替・タイマー操作）とカウントダウン更新
- `draw()`: 毎フレーム `datetime.datetime.now()` を取得し、ページに応じて描画
- `format_current_time(now, display_mode, include_seconds)`: 時刻文字列を生成
- `centered_text_x(text, char_width=8, screen_width=144)`: 中央揃えの x 座標を計算
- `run()`: `App()` をインスタンス化する関数

#### App クラスの状態

| 属性                  | 型      | 説明                                 |
|-----------------------|---------|--------------------------------------|
| `display_page`        | str     | `"clock"` または `"timer"`           |
| `display_mode`        | str     | `"time"` または `"date"`（時計画面） |
| `time_display_mode`   | str     | `"12h"` または `"24h"`              |
| `timer_set_seconds`   | int     | タイマー設定秒数（デフォルト 300）   |
| `timer_remaining`     | float   | 残り秒数                             |
| `timer_running`       | bool    | カウントダウン中かどうか             |
| `timer_done`          | bool    | タイマーが 0 に達したかどうか        |

### `timer.py`

`draw_timer(remaining, set_seconds, running, done, time_display_mode, font)`:

1. `math.ceil(remaining)` で表示用の残り時間（分・秒）を計算
2. `draw_face` で文字盤を描画
3. "TIMER" ラベルを (62, 24) に表示
4. 現在時刻を (y=84) に表示（フォント付き、グレー）
5. `draw_hands(..., show_hour=False)` で分針・秒針を描画
6. 残り時間テキストを画面下部中央に表示（状態に応じた色）

### `face.py`

`draw_face(cx, cy, radius)`:

1. `pyxel.circ(cx, cy, radius, 1)` — 濃紺の文字盤塗りつぶし
2. `pyxel.circb(cx, cy, radius, 2)` — 外ベゼル
3. `pyxel.circb(cx, cy, radius-1, 6)` — 内ベゼル
4. 時間目盛り 12本: `fraction = i/12`、`radius-3` → `radius-7` の線、色 6
5. 分目盛り 60本: `fraction = i/60`、5の倍数はスキップ、`radius-2` → `radius-4` の線、色 13
6. 中心ピボット: `pyxel.circ(cx, cy, 2, 11)`

### `hands.py`

`draw_hands(cx, cy, h, m, s, show_hour=True)`:

**フラクション計算（スムーズな動き）:**
```python
sec_fraction  = s / 60
min_fraction  = (m + s / 60) / 60
hour_fraction = ((h % 12) + m / 60) / 12
```

**針の仕様:**

| 針   | 先端半径 | 尾半径 | 色  | 太さ処理                  |
|------|---------|--------|-----|---------------------------|
| 時針 | 32 px   | -8 px  | 7   | cx+1 にオフセット線を追加 |
| 分針 | 48 px   | -10 px | 7   | cx+1 にオフセット線を追加 |
| 秒針 | 52 px   | -14 px | 8   | 1px のみ                  |

- `show_hour=False` の場合、時針の描画をスキップ（タイマーモード用）
- 描画順: 時針 → 分針 → 秒針（秒針が最前面）

## 注意点

- `angle_to_xy` の戻り値は `int(round(...))` で整数化している
- `datetime.now()` は `draw()` 内で毎フレーム呼ぶ（`update()` ではなく `draw()` で取得）
- `display_scale` は Pyxel 2.x から対応
- タイマーのカウントダウンは FPS=10 に依存（毎フレーム 0.1秒減算）
- 時計画面での点滅判定: `(pyxel.frame_count // 5) % 2 == 0`（FPS=10 のため 0.5秒間隔）
